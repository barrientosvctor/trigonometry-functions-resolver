# trigonometry-functions-resolver
A simple web to automate trigonometry functions answers.

## Requeriments
* [Node.js](https://nodejs.org/en)

## Run server

Type these commands:
```
$ npm i
$ node .
```

or

```
$ npm i
$ npm run start
```

Now, go to the browser of you choice and type the following url: `http://localhost:4000`

> Enjoy :)
