# trigonometry-functions-resolver
Web to automate trigonometry functions answers.

## Requeriments
* [Node.js](https://nodejs.org/en)

## Ways to run the web

### Way 1

Run these commands:
```
$ npm i
$ npm run server
```

### Way 2

Run these commands:
```
$ npm i
$ node .
```

or

```
$ npm i
$ npm run start
